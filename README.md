## 📋 О проекте

**AllergyDetect** — это веб-приложение для анализа фотографий продуктов с использованием AI-модели (Qwen3-VL:4B через Ollama). Приложение помогает пользователям с аллергиями и медицинскими противопоказаниями безопасно выбирать продукты, определяя потенциально опасные ингредиенты.

### 🎯 Основная цель

Обеспечить людям с пищевыми аллергиями и ограничениями по здоровью возможность быстро и точно оценить безопасность продукта перед употреблением.

## <a id="установка"></a> 🚀 Установка и запуск

### Требования

- Docker и Docker Compose
- Git
- ~8GB свободной оперативной памяти (для Ollama)

### 1️⃣ Клонирование репозитория

```bash
git clone https://github.com/MiniLynx13/FullStack.git
cd FullStack
```

### 2️⃣ Настройка переменных окружения

Создайте файл `.env` в корневой директории:

```bash
# MinIO
MINIO_ROOT_USER=grant_access
MINIO_SECRET_KEY=ai_food_analysing

# Backend
DATABASE_PATH=/app/data/app.db
ACCESS_TOKEN_EXPIRE_MINUTES=15
REFRESH_TOKEN_EXPIRE_DAYS=7
CORS_ORIGINS=http://localhost:3000
ENVIRONMENT=development

# Общие
COMPOSE_PROJECT_NAME=fullstack
```

### 3️⃣ Запуск Docker Compose

При первом запуске: `docker-compose up --build -d`

При повторных запусках: `docker compose up -d`

Это поднимет 4 контейнера:

- **minio** - хранилище изображений (порты: 9000, 9001)
- **ollama** - AI-модель (порт: 11434)
- **backend** - FastAPI приложение (порт: 8000)
- **frontend** - React приложение (порт: 3000)

### 4️⃣ Доступ к сервисам

| Сервис | URL |
|--------|-----|
| Frontend | http://localhost:3000 |
| Backend API | http://localhost:8000 |
| MinIO Console | http://localhost:9001 |
| API Docs (Swagger) | http://localhost:8000/docs |

### 5️⃣ Остановка всех контейнеров

`docker compose down`

### 6️⃣ Остановка с удалением томов (очистка данных)

`docker compose down -v`

## 🐛 Graceful Degradation

При недоступности Ollama или ошибках анализа, система автоматически переключается в **fallback-режим**:

1. **1-я попытка**: Обычный вызов Ollama
2. **2-я попытка**: Сжатие изображения до 1000px
3. **3-я попытка**: Сжатие до 500px
4. **Fallback**: Возврат заглушки с сообщением о временной недоступности

Это гарантирует, что пользователь никогда не увидит ошибку 500.

## 🔐 Безопасность

| Механизм | Описание |
|----------|----------|
| **Хеширование паролей** | SHA-256 |
| **JWT токены** | Access (15 мин) + Refresh (7 дней) |
| **Очистка токенов** | Автоматическое удаление просроченных |
| **Ролевая модель** | user / admin / banned |
| **CORS** | Настраиваемые разрешенные источники |
| **Fallback storage** | Локальное хранилище при недоступности MinIO |
